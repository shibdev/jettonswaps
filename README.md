# jettonswaps
Bot for detect jetton swaps (only swaps, not liquidity) at [DeDust.io](https://dedust.io/)

### Installing
```bash
git clone https://github.com/shibdev/jettonswaps
cd jettonswaps
pip3 install -r requirements.txt
cd src
```
